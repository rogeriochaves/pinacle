# Supermaven Code Completion [Pinacle provides built-in AI-powered code completion via Supermaven in VS Code.]

## Automatic Setup

VS Code in your pod is pre-configured with the Supermaven extension. When you first open VS Code, you'll see a notification prompting you to activate Supermaven.

## Activation Process

1. **Open VS Code** in your pod
2. **Wait for the toast notification** to appear (usually in the bottom-right corner)
3. **Click "Use Free Version"**
4. **Enter your email address** when prompted
5. **Code completion will activate** automatically

![Supermaven activation toast](/docs/supermaven.png)

## Usage

Once activated, Supermaven provides intelligent code suggestions as you type. The suggestions appear inline and can be accepted by pressing `Tab`.

Supermaven works across all programming languages and integrates seamlessly with your existing workflow.
